# Lokalpedia Library

Lokalpedia Library is a desktop-based library management application developed using **Java**, **JavaFX**, and **MySQL**.  
This application is designed to help manage library data efficiently and in an organized manner.

## Application Overview
Lokalpedia Library allows **guests** to browse the book catalog, search for titles, and check book stock availability.  
For **administrators**, the application provides full management features including book data, member data, borrowing and returning transactions, as well as report generation.

## Features
### Guest
- View book catalog
- Search books
- Check book stock availability

### Administrator
- Login & Register (Admin only)
- CRUD Books
- CRUD Members
- Borrowing & Returning Books
- Automatic Loan Records
- Export Reports to CSV (Books, Members, Borrowing Data)
- View and Manage Library Transactions

## Technologies Used
- Java  
- JavaFX  
- MySQL  
- CSS (JavaFX Styling)
